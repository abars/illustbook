CSSリファレンス

概要

このページでは、お絵描き掲示板のCSSをカスタマイズするための基本情報を提供します。CSSでは、一度しか出現しない要素にはID(#)、複数出現する要素にはCLASS(.)を割り当てています。

 

 

お絵かき掲示板のCSS構成

お絵描き掲示板は、次のようなCSS構成になっています。

 

#container

#header　お絵描き掲示板のタイトル、ロゴイラストを表示

#contents-header　イラストの新規作成やログインに使用するボタンを表示

#contents

#main_2col/#main_1col　2col表示の場合はmain_2col、1col表示の場合はmain_1colが選択される

#entry-head

#page-head　ページの一覧を表示

#rss　RSSへのリンクを表示

.entry　イラストを1枚表示

.entry　イラストを1枚表示

#entry-bottom

#page-bottom　ページの一覧を表示

#side　サイドバーを表示

以下、各項目のタイトル文字にはh4が指定されます

#free　フリーエリア

#category　カテゴリ一覧

#affiliate　アフィリエイト

#comment　コメント

#counter　カウンター

#menu　掲示板メニュー

#twitter


#contents-footer　コピーライトを表示

 

 

イラストのCSS構成

お絵描き掲示板内の各イラストは、次のようなCSS構成になっています。

 

.entry

.entry-title

h3　イラストタイトル

.clap　拍手


.entry-illust　イラスト

.entry-name　イラストの基本情報

.entry-comment　イラストのコメント

.entry-bottom　コメントを書く、コメントを読む
