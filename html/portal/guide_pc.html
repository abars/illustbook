{% if is_english %}Layer{% else %}レイヤー{% endif %}

{% if is_english %} Layer is a sheet of clear transparent film.

{% else %} レイヤーは透明の紙のようなもので、下のレイヤーから順に重なって表示されます。

レイヤーには以下の操作を行うことができます。

ウィンドウメニュー

ウィンドウメニューには、以下のサブメニューがあります。

 

ファイルメニュー(PC版)

新規クリアして新しいイラストを描き始めます。
開くパソコン上の画像ファイル(.jpg/.png/.ilb)を開きます。画像のアップロードにも使用できます。
保存.ilb形式で保存します。
JPG書出JPEGファイルを書き出します。(*)
PNG書出PNGファイルを書き出します。(*)

(*)保存メニューに統合したいのですが、バイナリを先に生成しなければならないというFlashの制約から別項目となっています

 

ファイルメニュー(アプリ版)

新規クリアして新しいイラストを描き始めます。
カメラロールから開く画像をカメラロールから読込みます。
カメラロールに保存画像をカメラロールに保存します。

 

キャンバスメニュー

キャンバスサイズ変更 キャンバスのサイズを変更します。 現在のキャンバスよりも小さなサイズを指定した場合は縮小されます。 大きなサイズを指定した場合は余白が追加されます。
全体を選択 選択ツールでキャンバス全体を選択します。
選択を解除 選択ツールで選択している範囲を解除します。
選択範囲を切り取り 選択ツールで選択している範囲を新たなキャンバスにします。
フィルタを適用画像に明るさとコントラストのフィルタを適用します。

 

編集メニュー

レイヤーを追加新しいレイヤーを上に追加します。
レイヤーをクリア選択しているレイヤーをクリアします。
レイヤーを上に複製選択しているレイヤーを複製して新しいレイヤーを作成します。
レイヤーを下に転写選択しているレイヤーと下のレイヤーを統合します。
レイヤーの透明色保護透明色を保護して描画するモードに切り替えます。線の色だけを変えたい場合などに使用します。
レイヤーを左右反転レイヤーの画像を左右反転します。
レイヤーを上下反転レイヤーの画像を上下反転します。
レイヤーに画像を読込レイヤーに外部の画像ファイルをインポートします。

 

設定メニュー

スポイトスポイトがレイヤーの色を取得するか、キャンバスの色を取得するかを選択します。
筆圧ツールごとに筆圧を使用するかどうかを選択します。
入抜き描画した線の始点と終点を細くするかを選択します。
手ぶれ補正手ぶれ補正を使用するかを選択します。
トーン1/2と1/4のトーンで塗るかどうかを選択します。
アンチエイリアスアンチエイリアスを切ることができます。
グリッドグリッドを表示することができます。投稿時はグリッドは表示されません。
UI配置UIの配置を4パターンから選択することができます。
テキストテキストツールの文字の方向を指定することができます。
フォントを選択テキストツールで使用するフォントを選択することができます。
{% endif %}
